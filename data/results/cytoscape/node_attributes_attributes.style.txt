
CYTOSCAPE STYLE GUIDE
====================
1. IMPORT NETWORK: File -> Import -> Network from File -> Select 'network_tumor_edges.csv'
2. IMPORT TABLE:   File -> Import -> Table from File -> Select 'node_attributes.csv'
   (Make sure Key Column maps to 'GeneName')

STYLE MAPPINGS:
- Node Fill Color: Map to 'Regulation' (Discrete) or 'DeltaBetweenness' (Continuous)
- Node Size: Map to 'EnsembleScore' (Continuous)
- Node Label: Map to 'GeneName' (Only for DiCERank < 20)
